HTML


O que é?

HTML (acrônimo para a expressão inglesa HyperTexto Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada para produzir páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Sobre o curso:

Este curso ensina desde os conceitos básicos da inguagem HTML, até a criação de formulários para a Web. ideal para inciantes, o par quem quer um guia de consulta. t bém possui um Glossário com todas as palavras mais utillizadas na Internet

DreamWeaver

O que é?

O Macromedia Dreamweaver é um software de criação para a web criada pela Macromedia (agora Adobe Systems), que está atualmente na versão CS3. Versões iniciais da aplicação serviam como um simples editor HTML WYSIWYG (What You See Is What You Get)

Flash


O que é?

O termo "Flash" costumava se referir a uma tecnologia da Adobe usada para criar animações interativas, jogos e sites dinâmicos. No entanto, a partir do final dos anos 2000 e início dos anos 2010, o uso do Flash começou a diminuir devido a problemas de desempenho, segurança e compatibilidade com dispositivos móveis. Como resultado, a Adobe anunciou que deixaria de suportar o Flash a partir do final de 2020.

Fireworks


O que é?

O Fireworks é uma ferramenta para desenvolvimento de Layouts web. Com o FW podemos criar e editar imagens para a WEB, além de animações simples.
